Features:

* Streamlit interface with chat history
* Sends user input to Gemini AI
* Displays AI responses dynamically
* Maintains conversation state across messages

Requirements:

* Python 3.10+
* streamlit
* gemini_client (custom or installed module)

Install dependencies:
pip install streamlit gemini_client

How to Run:
streamlit run app.py

Open the browser at [http://localhost:8501/](http://localhost:8501/).

Project Structure:

* app.py
* gemini_client.py
* README.txt

Description:
A simple AI chat app using Streamlit to communicate with Gemini AI in real-time. Keeps session messages and displays user and assistant messages.
